Project Description

The project aims to implement a digital open-source intelligence web application for information gathering on publicly available data relating to domains and online systems. The project incorporates multiple digital OSINT functions such as DNS record lookups, IP address lookups, WHOIS data, mail policies, etc into a single application using APIs.
A risk assessment logic with the use of a weighing system will identify any potential vulnerabilities for a domain searched on the application.
